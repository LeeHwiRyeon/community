## 🎯 소개

커뮤니티 플랫폼 2.0은 다음 4가지 소셜 기능을 제공합니다:

- 🤝 **팔로우 시스템**: 관심있는 사용자를 팔로우하고 소식을 받아보세요
- 💬 **멘션 시스템**: @사용자명으로 다른 사용자를 언급하세요
- 📤 **공유 기능**: 게시물을 소셜 미디어에 공유하세요
- 🚫 **차단 기능**: 원하지 않는 사용자를 차단하세요

## 🤝 팔로우 시스템

### 팔로우란?

팔로우는 특정 사용자의 활동을 구독하는 기능입니다. 팔로우한 사용자의 새 게시물을 피드에서 쉽게 볼 수 있습니다.

### 사용자 팔로우하기

#### 방법 1: 프로필 페이지에서

1. 팔로우하고 싶은 사용자의 프로필로 이동
2. **팔로우** 버튼 클릭
3. 버튼이 **팔로잉**으로 변경됨

```
[프로필 사진] @username
이름: 홍길동
소개: 개발자입니다.

[팔로우] ← 클릭
```

#### 방법 2: 게시물에서

1. 게시물 작성자 이름 옆의 **팔로우** 버튼 클릭
2. 즉시 팔로우 관계 성립

```
홍길동 @username [팔로우] ← 클릭
게시물 내용...
```

### 언팔로우하기

1. 팔로잉 중인 사용자의 프로필 또는 게시물 확인
2. **팔로잉** 버튼 클릭
3. 팔로우 관계 해제

```
[프로필 사진] @username
[팔로잉 ▼] ← 클릭 → [언팔로우]
```

### 팔로워/팔로잉 목록 보기

#### 내 팔로워 보기
1. 내 프로필 페이지 이동
2. **팔로워 50명** 클릭
3. 나를 팔로우하는 사용자 목록 확인

#### 내 팔로잉 보기
1. 내 프로필 페이지 이동
2. **팔로잉 30명** 클릭
3. 내가 팔로우하는 사용자 목록 확인

### 팔로우 추천

시스템이 자동으로 추천하는 사용자:
- 🔥 인기 사용자 (팔로워가 많은)
- 📝 활발한 사용자 (게시물이 많은)
- 🎯 비슷한 관심사를 가진 사용자

**위치**: 홈 화면 오른쪽 사이드바 > "추천 팔로우"

### 팔로우 알림

팔로우 관련 알림:
- ✅ 새로운 팔로워 알림
- 📢 팔로잉 사용자의 새 게시물 (선택)
- 🔔 맞팔로우 알림

**알림 설정**: 설정 > 알림 > 팔로우 알림

## 💬 멘션 시스템

### 멘션이란?

멘션은 게시물이나 댓글에서 특정 사용자를 언급하는 기능입니다. `@사용자명` 형식으로 작성합니다.

### 멘션 작성하기

#### 자동완성 기능 사용

1. 게시물 또는 댓글 입력창에 `@` 입력
2. 사용자명 입력 시작 (예: `@홍`)
3. 자동완성 목록에서 선택
4. Enter 또는 클릭으로 멘션 완성

```
게시물 작성:
@홍길동 님, 이 게시물 확인해주세요!
     ↑
  자동완성 목록:
  - 홍길동 @honggildong
  - 홍진호 @hongjinho
```

#### 키보드 단축키

- `↑` `↓`: 목록 이동
- `Enter`: 선택
- `Esc`: 자동완성 닫기

### 멘션 알림 확인

#### 알림 센터에서

1. 상단 네비게이션 바의 🔔 아이콘 클릭
2. "멘션" 탭 선택
3. 멘션 목록 확인

```
🔔 알림 (3)
[전체] [멘션] [팔로우]

💬 홍길동님이 회원님을 멘션했습니다.
   "@username 이 글 어떻게 생각하세요?"
   2분 전
```

#### 멘션 페이지에서

1. 프로필 메뉴 > "멘션" 클릭
2. 모든 멘션 목록 확인
3. 읽음/읽지 않음 필터 사용

### 멘션 관리

#### 읽음 표시
- 멘션 클릭 시 자동으로 읽음 처리
- 또는 "읽음으로 표시" 버튼 클릭

#### 전체 읽음 처리
- 멘션 목록 상단의 "모두 읽음" 버튼 클릭

#### 멘션 필터링
- **모두**: 모든 멘션
- **읽지 않음**: 아직 확인하지 않은 멘션
- **게시물**: 게시물에서의 멘션만
- **댓글**: 댓글에서의 멘션만

## 📤 공유 기능

### 게시물 공유하기

#### 소셜 미디어 공유

1. 게시물 하단의 **공유** 버튼 클릭
2. 공유할 플랫폼 선택:
   - 🐦 **Twitter**: 트윗으로 공유
   - 📘 **Facebook**: 페이스북 타임라인에 공유
   - 💼 **LinkedIn**: 링크드인 프로필에 공유
   - 📋 **링크 복사**: URL 클립보드에 복사

```
게시물 제목
게시물 내용...

[좋아요 5] [댓글 2] [공유 ▼] ← 클릭
                    ↓
              [🐦 Twitter]
              [📘 Facebook]
              [💼 LinkedIn]
              [📋 링크 복사]
```

#### Twitter 공유

1. 공유 메뉴에서 **Twitter** 선택
2. 트위터 창 열림 (트윗 미리 작성됨)
3. 필요시 내용 수정
4. **Tweet** 버튼 클릭

**공유 내용 예시**:
```
[게시물 제목] - Community Platform
https://community.example.com/posts/123
#CommunityPlatform
```

#### Facebook 공유

1. 공유 메뉴에서 **Facebook** 선택
2. 페이스북 공유 다이얼로그 열림
3. 코멘트 추가 (선택)
4. **공유하기** 버튼 클릭

#### LinkedIn 공유

1. 공유 메뉴에서 **LinkedIn** 선택
2. 링크드인 공유 창 열림
3. 코멘트 추가 (선택)
4. **게시** 버튼 클릭

#### 링크 복사

1. 공유 메뉴에서 **링크 복사** 선택
2. 자동으로 URL이 클립보드에 복사됨
3. "링크가 복사되었습니다!" 메시지 표시
4. 원하는 곳에 붙여넣기 (Ctrl+V)

### 공유 통계 확인

#### 게시물 공유 수
- 게시물 하단에 표시: "공유 15회"
- 클릭하면 상세 통계 확인

#### 상세 통계
```
📊 공유 통계
총 공유: 15회

플랫폼별:
🐦 Twitter: 8회
📘 Facebook: 5회
💼 LinkedIn: 2회
📋 링크 복사: 5회

최근 공유: 2분 전
```

### 인기 게시물 확인

1. 사이드바의 "인기 게시물" 섹션
2. 공유가 많은 게시물 순으로 표시
3. 실시간 업데이트

```
🔥 인기 게시물
1. [게시물 제목] - 공유 50회
2. [게시물 제목] - 공유 42회
3. [게시물 제목] - 공유 38회
```

## 🚫 차단 기능

### 차단이란?

차단은 특정 사용자와의 모든 상호작용을 차단하는 기능입니다.

#### 차단 시 효과
- ❌ 차단한 사용자의 게시물이 보이지 않음
- ❌ 차단한 사용자의 댓글이 보이지 않음
- ❌ 차단한 사용자가 내 프로필/게시물을 볼 수 없음
- ❌ 서로 팔로우 관계 자동 해제
- ❌ 멘션 불가능

### 사용자 차단하기

#### 프로필에서 차단

1. 차단할 사용자의 프로필 페이지 이동
2. ⋮ (더보기) 메뉴 클릭
3. **차단** 선택
4. 차단 확인 다이얼로그에서 **차단** 버튼 클릭

```
[프로필 사진] @username
[팔로우] [⋮ 더보기] ← 클릭
            ↓
        [신고하기]
        [차단] ← 선택
```

#### 게시물에서 차단

1. 게시물 우측 상단의 ⋮ 메뉴 클릭
2. **작성자 차단** 선택
3. 확인 다이얼로그에서 **차단** 버튼 클릭

#### 차단 확인 다이얼로그

```
⚠️ 사용자 차단

@username 님을 차단하시겠습니까?

차단된 사용자는 회원님의 게시물과 댓글을 볼 수 없으며,
회원님도 상대방의 콘텐츠를 볼 수 없습니다.

차단 이유 (선택사항):
[___________________________]

[취소] [차단]
```

### 차단 이유 입력 (선택)

차단 이유를 기록할 수 있습니다:
- 📝 나중에 차단 목록에서 확인 가능
- 🔒 차단된 사용자에게는 보이지 않음
- 📊 시스템 개선을 위한 참고 자료

**예시**:
- "스팸 게시물"
- "부적절한 댓글"
- "괴롭힘"

### 차단 해제하기

#### 차단 목록에서

1. 설정 > 개인정보 및 보안 > **차단한 계정**
2. 차단 해제할 사용자 찾기
3. **차단 해제** 버튼 클릭
4. 확인

```
🚫 차단한 계정 (3명)

[검색: ____________]

홍길동 @username
차단일: 2025-11-09
이유: 스팸 게시물
[차단 해제] ← 클릭
```

#### 프로필에서

1. 차단한 사용자의 프로필 페이지 이동 (URL 직접 입력)
2. **차단 해제** 버튼 클릭
3. 확인

### 차단 목록 관리

#### 검색 기능
- 차단 목록 상단의 검색창 사용
- 사용자명, 이름으로 검색

#### 정렬 옵션
- 최근 차단순
- 오래된 차단순
- 이름순

#### 페이지네이션
- 한 페이지에 10명씩 표시
- 하단의 페이지 번호로 이동

### 차단 통계

내 프로필 > 통계 > 차단 정보:
```
차단 통계
내가 차단한 사용자: 3명
나를 차단한 사용자: 1명
(나를 차단한 사용자 수는 확인만 가능)
```

## ❓ FAQ

### 팔로우 관련

**Q: 팔로우하면 상대방에게 알림이 가나요?**  
A: 네, 기본적으로 새 팔로워 알림이 전송됩니다. 상대방이 알림을 끈 경우 전송되지 않습니다.

**Q: 비공개 계정을 팔로우할 수 있나요?**  
A: 비공개 계정은 팔로우 요청을 보내고 승인을 받아야 합니다. (향후 기능)

**Q: 팔로우 수 제한이 있나요?**  
A: 하루 최대 100명까지 팔로우 가능합니다. (스팸 방지)

**Q: 서로 팔로우(맞팔)하면 무엇이 달라지나요?**  
A: 맞팔로우 시 프로필에 "서로 팔로우함" 표시되며, DM(향후 기능) 사용이 가능합니다.

### 멘션 관련

**Q: 한 게시물에 여러 명을 멘션할 수 있나요?**  
A: 네, 제한 없이 여러 명을 멘션할 수 있습니다. (예: @user1 @user2 @user3)

**Q: 멘션 알림을 끌 수 있나요?**  
A: 설정 > 알림 > 멘션 알림에서 끄거나 필터링할 수 있습니다.

**Q: 삭제된 사용자를 멘션하면?**  
A: 자동완성에 나타나지 않으며, 강제로 입력해도 알림이 전송되지 않습니다.

**Q: 멘션을 수정하거나 삭제할 수 있나요?**  
A: 게시물/댓글을 수정하여 멘션을 추가/제거할 수 있습니다.

### 공유 관련

**Q: 공유 횟수가 실시간으로 업데이트되나요?**  
A: 네, 실시간으로 업데이트됩니다.

**Q: 공유를 취소할 수 있나요?**  
A: 소셜 미디어에 이미 공유된 내용은 해당 플랫폼에서 직접 삭제해야 합니다.

**Q: 비공개 게시물도 공유할 수 있나요?**  
A: 비공개 게시물은 공유 버튼이 비활성화됩니다.

**Q: 공유한 사람을 확인할 수 있나요?**  
A: 현재는 공유 횟수만 확인 가능합니다. (개인정보 보호)

### 차단 관련

**Q: 차단한 사용자가 나를 볼 수 있나요?**  
A: 아니요. 차단 시 양방향으로 콘텐츠가 숨겨집니다.

**Q: 차단하면 과거 댓글도 사라지나요?**  
A: 네, 차단 즉시 해당 사용자의 모든 댓글이 숨겨집니다.

**Q: 차단한 사용자가 차단 사실을 알 수 있나요?**  
A: 직접적인 알림은 없지만, 내 프로필/게시물에 접근 시 제한 메시지가 표시됩니다.

**Q: 차단 해제 후 팔로우 관계는?**  
A: 자동으로 복구되지 않습니다. 다시 팔로우해야 합니다.

**Q: 차단 횟수 제한이 있나요?**  
A: 차단할 수 있는 사용자 수에는 제한이 없습니다.

## 🔧 문제 해결

### 팔로우 버튼이 작동하지 않음

**증상**: 팔로우 버튼 클릭 시 반응 없음

**해결 방법**:
1. 페이지 새로고침 (F5)
2. 로그인 상태 확인
3. 브라우저 캐시 삭제
4. 다른 브라우저에서 시도

### 멘션 자동완성이 나타나지 않음

**증상**: @입력 후 자동완성 목록이 뜨지 않음

**해결 방법**:
1. 입력창에 포커스 확인
2. JavaScript 활성화 확인
3. 브라우저 확장 프로그램 비활성화
4. 페이지 새로고침

### 공유 버튼 클릭 시 새 창이 열리지 않음

**증상**: 소셜 미디어 공유 시 팝업 차단

**해결 방법**:
1. 브라우저 팝업 차단 해제
2. 주소창의 🚫 아이콘 클릭
3. "이 사이트의 팝업 항상 허용" 선택

### 차단한 사용자의 콘텐츠가 계속 보임

**증상**: 차단 후에도 게시물이 보임

**해결 방법**:
1. 페이지 새로고침
2. 브라우저 캐시 삭제
3. 차단 목록에서 해당 사용자 확인
4. 로그아웃 후 재로그인

### 알림이 오지 않음

**증상**: 팔로우/멘션 알림 미수신

**해결 방법**:
1. 설정 > 알림에서 해당 알림 활성화 확인
2. 브라우저 알림 권한 허용
3. 이메일 알림 설정 확인
4. 스팸 폴더 확인

### 에러 메시지 해결

#### "자기 자신을 팔로우할 수 없습니다"
- 다른 사용자의 프로필로 이동 후 시도

#### "이미 팔로우한 사용자입니다"
- 페이지 새로고침 후 버튼 상태 확인

#### "존재하지 않는 사용자입니다"
- 사용자가 탈퇴했거나 계정이 정지됨
- URL 확인 후 재시도

#### "인증이 필요합니다"
- 로그인 후 다시 시도
- 세션이 만료된 경우 재로그인
