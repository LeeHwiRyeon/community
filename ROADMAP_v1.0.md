# 🗺️ Community Platform - 개발 로드맵 v1.0

**최종 업데이트**: 2025년 11월 9일

## 📋 개요

이 문서는 Community Platform v1.0 이후의 개발 계획을 정리한 로드맵입니다. Phase 1부터 Phase 4까지의 단계적 확장 계획과 세부 작업 항목을 포함합니다.

---

## 🎯 Phase 1: 소셜/콘텐츠 기능 확장

### 1.1 RPG 스타일 프로필 시스템 (게이미피케이션)

**목표**: 사용자 참여 유도 및 커뮤니티 로열티 증가

**제안 기능**:
- **레벨 시스템**: 게시물/댓글 수에 따른 레벨업 (예: Lv.1 초보자 → Lv.10 전문가)
- **배지/칭호**: 활동 기반 획득 (첫 게시물, 100개 댓글 등)
- **스탯 포인트**: 활동성, 도움됨 점수, 커뮤니티 기여도 표시
- **아바타 커스터마이징**: 기본 아바타 선택 또는 간단한 커스터마이징
- **프로필 카드**: RPG 스타일 카드 형태로 프로필 표시

**전략**:
- 최소 버전으로 시작 (레벨 + 배지만)
- 사용자 피드백 후 확장
- Reddit의 카마나 Stack Overflow의 배지 시스템 참고

**세부 작업 계획**:
- [ ] RPG-01: 기존 활동 로그를 수집해 경험치·배지 조건을 수치화하고 우선 적용할 규칙을 `docs/rpg-profile.md`로 문서화
- [ ] RPG-02: `users`, `user_stats`, `user_badges` 테이블 스키마와 연관 인덱스를 포함한 마이그레이션 초안 작성 및 리뷰 요청
- [ ] RPG-03: 경험치 적립/레벨업/배지 발급을 담당하는 도메인 서비스 계층 설계 (`server-backend/src/services/profile/`)
- [ ] RPG-04: 활동 이벤트 트리거(게시글 작성, 댓글, 좋아요 등)에서 RPG 서비스 호출하도록 이벤트 파이프라인 정비
- [ ] RPG-05: 프로필 카드/툴팁에서 사용할 `/api/profile/:userId` 조회 API와 `/api/profile/progress` 대시보드 API 구현
- [ ] RPG-06: 프론트엔드 `ProfileCard`/`UserTooltip` 컴포넌트에 레벨, 배지, 스탯 뱃지 UI 추가 및 반응형 디자인 검증
- [ ] RPG-07: 레벨업 및 배지 획득 시 토스트·인앱 알림 트리거 추가, 알림 센터와 연동
- [ ] RPG-08: 단위 테스트(서비스 로직), 통합 테스트(API), e2e 시나리오(레벨업 플로우) 작성
- [ ] RPG-09: 운영 지표 (주간 레벨업 수, 배지 획득 분포) 대시보드 위젯 초안 정의 및 로깅/메트릭 파이프라인 연결

## 🎯 Phase 2: 관리자 기능 추가

### 2.1 관리자 대시보드
- 실시간 통계 (사용자 수, 게시물 수, 트래픽)
- 차트/그래프 시각화
- 보고서 생성

### 2.2 사용자 권한 관리
- 역할 부여 (관리자/모더레이터/일반 사용자)
- 권한 설정 (게시물 삭제, 사용자 차단 등)

### 2.3 콘텐츠 모더레이션
- 게시물 신고/차단 시스템
- 자동 필터링 및 수동 검토 큐
- 삭제된 콘텐츠 관리: 소프트 삭제, 링크 참조 제거, 감사 로그 기록

### 2.4 블랙리스트 관리
- 사용자/IP 차단
- 키워드 필터링
- 경고/정지 시스템

### 2.5 관리 쟁점 및 해결 방안

**주요 쟁점**:
- 스팸/욕설/하트스픽 처리
- 저작권 침해 방지
- 사용자 간 갈등 (하라링, 스토킹)
- 콘텐츠 정합성 유지
- 사용자 활동 모니터링

**해결 전략**:
- 자동 필터링 도구 및 AI 기반 감지
- 저작권 체크 시스템 (이미지 해시 비교)
- 신고 및 제재 프로세스 명확화 (가이드라인 제공)
- 커뮤니티 자율 규제 (모더레이터 임명, 포인트 시스템)
- 투명한 정책 공개 및 피드백 채널 운영

## 🎯 Phase 2-3: 커뮤니티 보드 기획

### 3.1 보드 구조
- 일반 커뮤니티 (자유/유머/잡담)
- 게임 관련 (뉴스, 리뷰, 공략, e스포츠)
- 방송/엔터테인먼트 (리뷰, 편성표, 토크)
- 이벤트/모임 (오프라인 모임, 대회, 챌린지)
- 창작/코스프레 (작품 공유, 튜토리얼, 갤러리)

### 3.2 게시판 세부 기능

#### 공통 요소
- 정렬 옵션 (최신순, 인기순, 댓글순)
- 필터링 (태그, 작성자, 기간)
- 게시물 미리보기 카드 (썸네일, 요약)
- 북마크 및 공유
- 신고 버튼 및 모더레이션 큐

#### 인기 게시판
- 트렌딩 알고리즘 (일간/주간 인기글)
- 추천 포스트 (관리자/모더레이터 큐레이션)
- 사용자 배지 연동 (인기글 작성자 강조)

#### 공략/가이드 게시판
- 가이드 템플릿 (섹션 구조 제공)
- 난이도/소요시간 태그
- 동영상/이미지 슬라이드 지원
- 팁/변경사항 업데이트 로그

#### 리뷰 게시판

- **인디게임 소개 패턴화 시스템**: 게임 소개 콘텐츠 표준 템플릿

- **일반 뉴스 내용 업데이트 시스템**: 콘텐츠 유지보수 및 자동화

#### 뉴스 탭 UI/UX 설계
- **탭 구조**: 실시간 뉴스 / 아카이브 / 인터뷰 / 게임 소개
