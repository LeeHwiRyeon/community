# UI 트리뷰 시스템

## 📋 개요
Community Platform v1.3의 UI 트리뷰 시스템입니다. 폴더, 사용자, 콘텐츠 등을 계층적 구조로 시각화하고 태그 시스템을 통해 효율적으로 관리합니다.

## 🌳 트리뷰 기본 구조

### 전체 트리 구조
```
┌─────────────────────────────────────────────────────────────────┐
│                    UI 트리뷰 시스템                             │

│  │   └── 👤 관리자 (태그: admin, system)                       │
│  ├── 📁 커뮤니티                                               │
```
