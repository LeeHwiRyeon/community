# Community Platform v1.3 시각화 시스템

## 📋 개요
Community Platform v1.3의 시각화 시스템입니다. 실제 다이어그램과 이미지를 통해 시스템 구조와 기능을 직관적으로 표현합니다.

## 📊 데이터 플로우 다이어그램

### 실시간 채팅 시스템 플로우
```
사용자 로그인
       ↓
   인증 확인
       ↓
  채팅방 입장
       ↓
  메시지 전송
       ↓
   암호화 처리
       ↓
  WebSocket 전송
       ↓
  서버 브로드캐스트
       ↓
  수신자에게 전달
       ↓
   복호화 처리
       ↓
  메시지 표시
```

### 파일 공유 시스템 플로우
```
파일 선택
       ↓
  드래그 앤 드롭
       ↓
  파일 암호화
       ↓
  서버 업로드
       ↓
  메타데이터 저장
       ↓
  다운로드 URL 생성
       ↓
  채팅방에 공유
       ↓
  수신자 다운로드
```
