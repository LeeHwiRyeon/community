# 👥 Community Platform v1.1 - 사용자 가이드

**Community Platform v1.1**을 처음 사용하시는 분들을 위한 완벽한 가이드입니다. 단계별로 따라하시면 쉽게 시작하실 수 있습니다.

## 🚀 **빠른 시작 가이드**

### **1단계: 첫 방문**
1. **웹사이트 접속**: `http://localhost:5002`
2. **메인 페이지**: 커뮤니티 개요 및 최신 소식 확인
3. **회원가입/로그인**: 우상단 버튼 클릭

### **2단계: 프로필 설정**
1. **프로필 페이지** 이동
2. **기본 정보** 입력 (닉네임, 소개)
3. **관심사 선택** (게임, 뉴스, 커뮤니티 등)
4. **알림 설정** 구성

### **3단계: 커뮤니티 탐색**
1. **게시판 둘러보기**: 다양한 주제별 게시판 확인
2. **인기 게시글**: 트렌딩 콘텐츠 확인
3. **검색 기능**: 관심 있는 주제 검색

## 🎮 **주요 기능 사용법**

### **📝 게시글 작성**
```
1. 게시판 선택
2. "새 글 쓰기" 버튼 클릭
3. 제목과 내용 작성
4. 태그 추가 (선택사항)
5. "게시" 버튼 클릭
```

### **💬 댓글 및 소통**
- **댓글 작성**: 게시글 하단에서 의견 남기기
- **좋아요/싫어요**: 게시글 평가하기
- **공유하기**: SNS나 링크로 공유
- **신고하기**: 부적절한 콘텐츠 신고

### **🎯 개인화 기능**
- **북마크**: 관심 있는 게시글 저장
- **팔로우**: 좋아하는 작성자 팔로우
- **알림 설정**: 맞춤형 알림 받기
- **다크모드**: 테마 변경

## 🎮 **게임 센터 이용법**

### **🕹️ 게임 플레이**
1. **게임 센터** 메뉴 클릭
2. **게임 선택**: Snake, Tetris, Pong 등
3. **게임 시작**: 간단한 조작법 확인 후 플레이
4. **점수 기록**: 자동으로 리더보드에 등록

### **🏆 리더보드 및 업적**
- **순위 확인**: 게임별 상위 랭커 확인
- **업적 달성**: 다양한 도전 과제 완료
- **포인트 적립**: 게임 플레이로 포인트 획득

## 💬 **채팅 시스템**

### **🗨️ 실시간 채팅**
1. **채팅방 입장**: 관심 주제별 채팅방 선택
2. **메시지 전송**: 실시간으로 소통
3. **이모티콘**: 다양한 이모티콘 사용
4. **파일 공유**: 이미지, 문서 공유 가능

### **👥 그룹 채팅**

## 🗳️ **투표 시스템**

### **📊 투표 참여**

2. **선택지 선택**: 원하는 옵션 클릭
3. **투표 완료**: 실시간 결과 확인
4. **의견 남기기**: 투표 이유 댓글로 공유

### **📝 투표 생성**
