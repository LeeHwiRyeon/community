## 1. 시작하기

### 1.1 플랫폼 소개

커뮤니티 플랫폼은 사용자들이 자유롭게 소통하고 정보를 공유할 수 있는 종합 커뮤니티 서비스입니다.

**주요 기능**:
- 📝 게시판: 다양한 주제의 게시글 작성 및 공유
- 💬 채팅: 실시간 DM 및 그룹 채팅
- 🔔 알림: 실시간 활동 알림
- 🔍 검색: 강력한 통합 검색 기능
- 👤 프로필: 개인화된 프로필 및 활동 관리
- 🎨 테마: 라이트/다크 모드 지원
- 🌐 다국어: 한국어/영어 지원

### 1.2 회원가입

1. 홈페이지 접속 (http://localhost:5002)
2. 우측 상단 "회원가입" 버튼 클릭
3. 필수 정보 입력 (이메일, 비밀번호, 사용자명)
4. 이메일 인증 완료

### 1.3 로그인

- 이메일과 비밀번호로 로그인
- 로그인 유지: 24시간 (기본) / 30일 (체크 시)
